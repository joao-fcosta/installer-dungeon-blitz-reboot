# <p align="center">Dungeon Blitz Reboot</p>
Este é o instalador e launcher oficial (não-oficial) para o **Dungeon Blitz Reboot**. Ele automatiza toda a configuração necessária, incluindo o download do motor Flash, do servidor de jogo e do ambiente Python necessário para rodar tudo localmente no seu Windows.

## 🚀 Como Jogar

1. **Baixe o executável:**
   [:arrow_down: **Download DungeonBlitz.exe**](https://github.com/joao-fcosta/installer-dungeon-blitz-reboot/raw/main/DungeonBlitz.exe)

2. **Execute o arquivo:**
   - Ao abrir o `DungeonBlitz.exe`, ele verificará se você já tem os arquivos necessários.
   - Caso seja a primeira vez, ele baixará automaticamente o Flash Player, o Servidor e um ambiente Python isolado.

3. **Aguarde a inicialização:**
   - O Launcher iniciará o servidor local e, em seguida, abrirá a janela do jogo.
   - **Não feche o Launcher** enquanto estiver jogando, pois ele gerencia o servidor em segundo plano.
